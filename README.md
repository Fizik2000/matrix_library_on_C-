# matrix_library_on_C++
Library for different mathematical with matrixes operations on C++
  
## Технологии
**Стек технологий**  
Язык программирования С++  
Makefile  
CMake  
Тестирование  
  
**Инструменты**  
Git, GitHub, Gtest, gcov, lcov  
  
## Установка и запуск  
Создание библиотеки - make s21_matrix_oop.a  
Тестирование - make test  
Проверка на покрытие кода - make gcov_report  
Проверка на стиль кода google - make clang_check  
Приведение к стилю кода google - make clang  
Удаление объяктных файлов - make clean
